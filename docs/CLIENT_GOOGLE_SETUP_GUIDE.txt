2-1. Google Cloud Console 권한
-----------------------------

목적: 구글 로그인(OAuth) 설정을 위해 필요합니다.

요청 권한:
  - 역할(Role): Editor 이상 (Editor 또는 Owner)

초대 방법(요약):
  1) https://console.cloud.google.com 접속
  2) 상단 프로젝트 선택 → Web3Star용 프로젝트를 선택 (또는 새로 생성)
  3) 왼쪽 메뉴 "IAM 및 관리자" (IAM) 클릭
  4) "멤버 추가" 버튼 클릭
  5) jsmh8292@gmail.com 입력
  6) 역할(Role)을 "Editor" 또는 "Owner"로 선택
  7) 저장

이렇게 하면 제가:
  - OAuth 동의 화면
  - OAuth 클라이언트 ID/Secret
을 직접 생성/설정할 수 있습니다.


2-2. Google AdMob 권한
----------------------

목적: 앱 안에 들어갈 실제 광고(AdMob)를 연결하기 위해 필요합니다.

요청 권한:
  - 역할(Role): Admin (관리자) 권한 추천

초대 방법(요약):
  1) https://admob.google.com 접속
  2) 상단 톱니바퀴(설정) → "계정" 또는 "사용자 및 권한"(Users) 메뉴 진입
  3) "사용자 추가" 또는 "Invite user" 버튼 클릭
  4) jsmh8292@gmail.com 입력
  5) 권한을 "관리자(Admin)"로 설정
  6) 초대 완료

이렇게 하면 제가:
  - Web3Star 앱을 AdMob에 등록
  - 광고 단위(Interstitial, Rewarded 등)를 생성
  - 생성된 광고 단위 ID를 앱 코드에 연결
을 대신 처리할 수 있습니다.
