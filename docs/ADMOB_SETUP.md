# Google AdMob 설정 가이드 (초보자용)

앱에 광고를 넣으려면 **Google AdMob**에서 앱을 등록하고, **앱 ID**와 **광고 단위 ID**를 받아서 프로젝트에 넣어야 합니다.  
아래 순서대로 **한 단계씩** 따라 하세요.

---

## 1. AdMob 들어가기

1. 브라우저에서 **https://admob.google.com** 접속
2. **Google 계정**으로 로그인 (없으면 계정 만들기)
3. 처음이면 "AdMob에 오신 것을 환영합니다" 화면이 나올 수 있음 → **시작하기** 버튼 클릭
4. **동의 및 계속** 등 약관에 동의하고 진행

---

## 2. 앱 추가하기 (Web3Star 앱 등록)

1. 왼쪽 메뉴에서 **앱** 클릭 (또는 상단 **앱**)
2. **앱 추가** 버튼 클릭
3. **플랫폼 선택**
   - **Android** 선택 후 **다음** 클릭
4. **스토어 등록 여부**
   - "Google Play에 앱이 등록되어 있나요?" → **아니요** 선택 (아직 스토어에 올리지 않았으면)
   - **다음** 클릭
5. **앱 정보 입력**
   - **앱 이름**: `Web3Star` (또는 원하는 이름)
   - **다음** 클릭
6. **등록 완료**
   - 화면에 **앱 ID**가 나오면 복사해서 메모장에 붙여넣어 두세요.  
     형식: `ca-app-pub-XXXXXXXXXXXXXXXX~XXXXXXXXXX`  
   - **바로 "광고 단위 추가" 화면으로 넘어가서 앱 ID가 안 보이면** → 아래 "앱 ID 찾는 방법" 참고.

---

### 앱 ID가 안 보일 때 (바로 광고 단위 추가로 넘어가는 경우)

1. AdMob 왼쪽 메뉴에서 **앱** 클릭.
2. **앱 목록**에 방금 만든 앱(예: Web3Star)이 보이면 **앱 이름**을 클릭해서 들어갑니다.
3. 앱 화면 안에서:
   - **앱 정보** / **앱 설정** / **설정** 같은 메뉴를 찾아 클릭하거나,
   - 화면 **상단**이나 **기본 정보** 영역에 **"앱 ID"** 또는 **"App ID"** 라고 적힌 부분을 찾습니다.
4. 그 옆에 있는 긴 값(`ca-app-pub-xxxx~yyyy` 형태, **물결(~)** 포함)이 **앱 ID**입니다.  
   → 복사해서 메모장에 붙여넣어 두세요.
5. 일부 화면에서는 **앱 이름 아래** 작은 글씨로 앱 ID가 나오기도 합니다.  
   목록에서 앱 한 줄을 클릭해 보거나, **세 개 점(⋮)** 메뉴 → **앱 정보** 등으로 들어가 보세요.

**정리**: "광고 단위 추가"는 그대로 진행해도 됩니다. **전면 광고**랑 **보상형 광고** 두 개 만든 다음, 나중에 **앱** 메뉴 → 해당 앱 클릭 → 앱 정보/설정에서 **앱 ID**만 복사하면 됩니다.

---

## 3. 광고 단위 만들기 (전면 광고 – 채굴 버튼용)

1. 방금 만든 **Web3Star** 앱이 보이는 목록에서 **앱 이름**을 클릭해서 들어갑니다.
2. **광고 단위** 탭이 보이면 클릭 (또는 왼쪽에서 **앱** → 해당 앱 → **광고 단위**)
3. **광고 단위 추가** 버튼 클릭
4. **광고 형식 선택**
   - **전면 광고(인터스티셜)** 선택
   - **다음** 클릭
5. **광고 단위 이름**
   - 예: `채굴 전면 광고` (구분하기 쉬운 이름 아무거나)
   - **광고 단위 만들기** 클릭
6. 완료되면 **광고 단위 ID**가 나옵니다.  
   형식: `ca-app-pub-XXXXXXXXXXXXXXXX/XXXXXXXXXX`  
   **이 값 복사해서 메모장에 붙여넣기.**  
   (예: `채굴 버튼 → 광고 시청` 할 때 쓰는 ID)

---

## 4. 광고 단위 하나 더 만들기 (보상형 광고 – Get More Point 슬롯용)

1. 같은 앱 안에서 다시 **광고 단위 추가** 클릭
2. **광고 형식 선택**
   - **보상형** 선택 (동영상 보상형)
   - **다음** 클릭
3. **보상형 광고 설정**
   - **보상 항목**은 그대로 두거나, 예: `포인트`, `1` 등 (원하면 수정)
   - **광고 단위 이름** 예: `Get More Point 슬롯`
   - **광고 단위 만들기** 클릭
4. 완료되면 **보상형 광고 단위 ID**가 나옵니다.  
   형식: `ca-app-pub-XXXXXXXXXXXXXXXX/XXXXXXXXXX`  
   **이 값도 복사해서 메모장에 붙여넣기.**

---

## 5. 프로젝트에 ID 넣기 (3곳)

아래에서 `[여기에 붙여넣기]` 부분을 **복사해 둔 값**으로 바꾸면 됩니다.

### 5-1. Android 앱 ID (한 번만)

- **파일**: `android/app/src/main/res/values/strings.xml`
- **찾을 내용**:  
  `<string name="admob_app_id">ca-app-pub-3940256099942544~3347511713</string>`
- **바꿀 내용**:  
  `<string name="admob_app_id">[2단계에서 복사한 앱 ID]</string>`  
  예:  
  `<string name="admob_app_id">ca-app-pub-1234567890123456~1234567890</string>`

**앱 ID**는 `~`(물결)이 하나 들어가고, **광고 단위 ID**는 `/`(슬래시)가 들어갑니다. 여기에는 **앱 ID**만 넣습니다.

---

### 5-2. 전면 광고 단위 ID (채굴 버튼)

- **파일**: `src/lib/admob.ts`
- **찾을 내용**:  
  `const INTERSTITIAL_AD_ID = 'ca-app-pub-3940256099942544/1033173712';`
- **바꿀 내용**:  
  `const INTERSTITIAL_AD_ID = '[3단계에서 복사한 전면 광고 단위 ID]';`  
  예:  
  `const INTERSTITIAL_AD_ID = 'ca-app-pub-1234567890123456/1111111111';`

---

### 5-3. 보상형 광고 단위 ID (Get More Point 슬롯)

- **파일**: 같은 파일 `src/lib/admob.ts`
- **찾을 내용**:  
  `const REWARDED_AD_ID = 'ca-app-pub-3940256099942544/5224354917';`
- **바꿀 내용**:  
  `const REWARDED_AD_ID = '[4단계에서 복사한 보상형 광고 단위 ID]';`  
  예:  
  `const REWARDED_AD_ID = 'ca-app-pub-1234567890123456/2222222222';`

---

### 5-4. (선택) 실제 배포할 때 테스트 끄기

- **파일**: `src/lib/admob.ts`
- **찾을 내용**:  
  `isTesting: true`
- **바꿀 내용**:  
  스토어에 올린 뒤 **실제 광고**를 보여주려면  
  `isTesting: false` 로 바꾸거나, 해당 줄을 지우면 됩니다.  
  (테스트할 때는 `true`로 두면 Google에서 제공하는 테스트 광고가 나옵니다.)

---

## 6. 정리 체크리스트

- [ ] **1단계**: https://admob.google.com 접속, Google 로그인
- [ ] **2단계**: 앱 추가 → Android → 앱 이름 입력 → **앱 ID** 복사
- [ ] **3단계**: 전면 광고 단위 추가 → **전면 광고 단위 ID** 복사
- [ ] **4단계**: 보상형 광고 단위 추가 → **보상형 광고 단위 ID** 복사
- [ ] **5-1**: `android/.../res/values/strings.xml` 에 **앱 ID** 넣기
- [ ] **5-2**: `src/lib/admob.ts` 에 **전면 광고 단위 ID** 넣기
- [ ] **5-3**: `src/lib/admob.ts` 에 **보상형 광고 단위 ID** 넣기
- [ ] **5-4**: (배포 시) `isTesting: false` 로 변경

---

## 7. ID만 구분해서 기억하기

| 구분 | 형식 예시 | 넣는 곳 |
|------|-----------|---------|
| **앱 ID** | `ca-app-pub-xxxx~yyyy` (물결 **~**) | `strings.xml` 의 `admob_app_id` |
| **전면 광고 ID** | `ca-app-pub-xxxx/yyyy` (슬래시 **/**) | `admob.ts` 의 `INTERSTITIAL_AD_ID` |
| **보상형 광고 ID** | `ca-app-pub-xxxx/zzzz` (슬래시 **/**) | `admob.ts` 의 `REWARDED_AD_ID` |

---

## 8. 수정 후 할 일

1. **저장** 후 `npm run build` 실행
2. `npx cap sync` 실행
3. Android Studio에서 앱 다시 실행해서 광고가 나오는지 확인

여기까지 하면 AdMob에서 할 일은 끝입니다.  
궁금한 단계가 있으면 "2단계에서 앱 추가가 안 보여요"처럼 **몇 단계에서 무엇이 안 되는지** 알려주면 그 부분만 더 자세히 설명해 줄 수 있습니다.
